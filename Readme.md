# 🛠️ My Terminal & Tools Setup
*By ACF*

Este repositorio contiene mi configuración personal de terminal y las herramientas de línea de comandos que utilizo para optimizar mi flujo de trabajo en análisis de datos.

---

## 🏛️ Comandos del Sistema (Unix Classics)
Los cimientos de la terminal. Útiles para navegación rápida.

* **`cd`**: Cambiar de directorio (Navegación).
* **`ls`**: Listar archivos (Exploración).
* **`rm`**: Borrar archivos (**Cuidado**: Acción irreversible).
* **`sudo`**: Ejecutar con permisos de administrador.
* **`man`**: Ver el manual completo de un comando.

---

## 🚀 Mejoras Pro (Homebrew Formulae)
Herramientas instaladas para hacer la terminal más humana y visual.

| Comando | Reemplaza a... | Por qué lo uso |
| :--- | :--- | :--- |
| **`eza`** | `ls` | Muestra iconos, colores y carpetas en cascada. |
| **`bat`** | `cat` | Visualiza código (SQL, DAX, JS) con colores. |
| **`tealdeer`** | `man` | Da ejemplos rápidos de uso sin leer manuales largos. |
| **`topgrade`** | (Varios) | Actualiza todo mi sistema con una sola palabra. |
| **`fd`** | `find` | Encuentra archivos en segundos por nombre. |
